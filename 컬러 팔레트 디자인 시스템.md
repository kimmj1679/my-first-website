## 컬러 사용 가이드

### Primary (`#FF0000` 계열)
| 변수 | 용도 |
|------|------|
| `--color-primary` | CTA 버튼, 브랜드 로고, 강조 태그 |
| `--color-primary-light` | 호버 상태, 활성 인디케이터 |
| `--color-primary-dark` | 버튼 클릭(active) 상태 |

> 전체 화면의 **5% 이하**로만 사용 — 많이 쓸수록 긴장감이 떨어짐

### Background (`#0F0F0F` 계열)
| 변수 | 용도 |
|------|------|
| `--color-bg-primary` | `<body>` 기본 배경 |
| `--color-bg-secondary` | 카드, 모달, 패널 배경 |
| `--color-bg-tertiary` | 사이드바, 드롭다운, 호버 상태 배경 |

### Text (`#FFFFFF` 계열)
| 변수 | 용도 |
|------|------|
| `--color-text-primary` | 제목(h1~h3), 주요 콘텐츠 |
| `--color-text-secondary` | 부제목, 채널명, 날짜 |
| `--color-text-muted` | 비활성 텍스트, placeholder |

### Accent (`#3EA6FF`)
| 변수 | 용도 |
|------|------|
| `--color-link` | 일반 링크, 해시태그 |
| `--color-link-hover` | 링크 호버 상태 |

## 반응형 고려사항

### 다크 모드 (기본값)
이 팔레트 자체가 다크 모드 기준으로 설계되어 있으므로, `prefers-color-scheme: dark` 환경에서 그대로 사용합니다.

### 라이트 모드 대응
```css
@media (prefers-color-scheme: light) {
  :root {
    --color-bg-primary: #FFFFFF;
    --color-bg-secondary: #F2F2F2;
    --color-bg-tertiary: #E5E5E5;
    --color-text-primary: #0F0F0F;
    --color-text-secondary: #606060;
    --color-text-muted: #909090;
    --color-border: #CCCCCC;
  }
}
```

### 접근성 (WCAG 기준)
| 조합 | 대비율 | 등급 |
|------|--------|------|
| `#FFFFFF` on `#0F0F0F` | 19.6 : 1 | AAA ✅ |
| `#AAAAAA` on `#0F0F0F` | 7.2 : 1 | AAA ✅ |
| `#3EA6FF` on `#0F0F0F` | 6.8 : 1 | AA ✅ |
| `#717171` on `#0F0F0F` | 4.6 : 1 | AA ✅ |

### 모바일 대응 팁
- 터치 버튼은 최소 `44px × 44px` 확보, `--color-button-primary` 적용
- 작은 화면에서는 `--color-text-muted` 텍스트 사용 최소화 → 가독성 저하 방지
- 썸네일/이미지 위 텍스트는 반드시 `--color-text-primary(#FFFFFF)` + `text-shadow` 병행 사용
